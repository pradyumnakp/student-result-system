# Student Result Management System

This project calculates student marks and generates an Excel report.

## Features
- Multiple subjects
- FA, SA calculations
- Semester calculation
- Final percentage
- Excel output using openpyxl

## Technologies Used
- Python
- openpyxl

## How to Run
1. Install openpyxl:
   pip install openpyxl

2. Run:
   python main.py
